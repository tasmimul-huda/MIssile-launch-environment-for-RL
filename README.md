# MIssile-launch-environment-for-RL
- **create virtual environment** 
`virtualenv myenv`
- **activate myenv**
`myenv\Scripts\activate`
- **Run**
`pip install -r requirements.txt`
- **Run DQN agent** 
`python main.py`
- **Run Q-learning agent** 
`python main_q.py`
